FIT.AI — Treino com Consistência

Aplicação full-stack mobile-first para treinar com consistência: treino do dia, acompanhamento visual de progresso (heatmap/streak) e um coach com IA. Desenvolvi o front (Next.js) e a API (Fastify) de ponta a ponta.

Destaques Técnicos

  • Front em Next.js (App Router) com Server Components validando sessão no servidor e fluxo de onboarding/plano ativo.
  • API REST em Fastify + TypeScript com arquitetura em camadas (routes/usecases/schemas), Prisma e PostgreSQL.
  • Coach virtual na rota /ai com AI SDK + Google Gemini, capaz de montar planos de treino via tools.
  • Login Google com Better Auth e sessão por cookie compartilhada entre front e API; schemas validados com Zod.
